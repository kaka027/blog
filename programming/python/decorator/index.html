    装饰器

    01. 装饰器的定义和语法

    装饰器本质是一个函数。它可以让其他函数在不需要做任何代码变动的前提下增加额外功能，装饰器的返回值也是一个函数对象。

    它经常用于有切面需求的场景，比如：插入日志、性能测试、事务处理、缓存、权限校验等场景。

    装饰器的语法很固定，如下：

    • 先定义一个装饰器（帽子）
    • 再定义业务函数或类（人）
    • 最后把装饰器（帽子）扣在函数或类（人）的头上。
    # 定义装饰器
def decorator(func):
    def wrapper(*args,**kwargs):
        return func()
    return wrapper()

# 定义业务函数并装饰
@decorator
def function():
    print("hello decorator")

    02. 普通装饰器

    以下实例演示一个普通的装饰器，满足的需求是：

    • 在函数执行前，先记录一行日志
    • 在函数执行后，再记录一行日志
    # 定义装饰器，参数func是被装饰的函数
def decorator(func):
    def wrapper(*args, **kwargs):
        print("{}函数运行前的日志，我要开始执行了".format(func.__name__))
        func(*args, **kwargs)  # func()表示函数执行，真正执行的业务函数也是这句代码
        print("{}函数执行后的日志，我执行结束了".format(func.__name__))

    return wrapper

    假如我的业务函数是，计算两个数字之和。再定义函数前加上装饰器的帽子

    @decorator
def add(x, y):
    print("{} + {} = {}".format(x, y, x + y))

    假设需要计算数字10 和 5 的求和，执行一下add 函数

    add(10,5)
>>>
add函数运行前的日志，我要开始执行了
10 + 5 = 15
add函数执行后的日志，我执行结束了

    03. 带参数的装饰器

    普通装饰器，函数内部只能实现单一功能，如果需要对不同的值进行不同的处理，就需要运用到带参数装饰器

    • 以下实例演示，来自不同国家的人，用不同语言打招呼
    • 首先定义装饰器函数，带参数的装饰器，需要以双层函数嵌套，在最外层函数留形参，传参方式和函数传参一样，使用位置传参
    # 定义带参数的装饰器
def say_hello(contry):
    def wrapper(func):
        def deco(*args, **kwargs):
            if contry == "china": # 判断国家并以对应的语言问候
                print("你好")
            elif contry == "america":
                print("hello")
            return func(*args, **kwargs) # 执行被装饰函数的代码

        return deco

    return wrapper
    • 装饰来自不同国家和其语言介绍的函数，并在装饰器内传递contry 实参
    # 小明，中国人
@say_hello("china")
def xiaoming():
    print("来自中国的问候")

@say_hello("america")
def jack():
    print("greetings from america")
    • 运行对应的函数，程序会执行装饰器内的判断，以对应语言问候，且会执行被装饰的函数
    xiaoming()
print("-----")
jack()

>>>
你好
来自中国的问候
-----
hello
greetings from america

    04. wraps装饰器

    使用装饰器后，由于装饰器返回了wrapper函数，替换了say_hello 函数，导致函数内__name__ （函数名）和__doc__ （函数说明文档）等属性变成了wrapper 函数的了。

    # 定义带参数的装饰器
def say_hello(contry):
    def wrapper(func):
        def deco(*args, **kwargs):
            """doc of deco"""
            if contry == "china":
                print("你好")
            elif contry == "america":
                print("hello")
            return func(*args, **kwargs)

        return deco

    return wrapper

# 小明，中国人
@say_hello("china")
def xiaoming():
    """doc of xiaoming"""
    print("来自中国的问候")

@say_hello("america")
def jack():
    print("greetings from america")

print(xiaoming.__name__)
print(xiaoming.__doc__)
>>>
deco
doc of deco

    如果想要显示say_hello 函数的name属性和doc属性，解决这一问题的办法是通过

    functools 模块下的wraps 装饰器。

    from  functools import wraps
# 定义带参数的装饰器
def say_hello(contry):
    def wrapper(func):
        @wraps(func)  #使用wraps装饰器
        def deco(*args, **kwargs):
            """doc of deco"""
            if contry == "china":
                print("你好")
            elif contry == "america":
                print("hello")
            return func(*args, **kwargs)

        return deco

    return wrapper

# 小明，中国人
@say_hello("china")
def xiaoming():
    """doc of xiaoming"""
    print("来自中国的问候")

@say_hello("america")
def jack():
    print("greetings from america")

print(xiaoming.__name__)
print(xiaoming.__doc__)
>>>
xiaoming
doc of xiaoming
