    面向对象

    01. 面向对象

    面向对象编程（Object-oriented Programming，简称 OOP），是一种封装代码的方法。

    将数据扔进列表，代码块扔进函数，都是封装的简单形式。面向对象本质也是一种封装，模拟真实世界的实物，将数据和函数封装进一个对象，不同的对象各司其职，完成最终需求的一种编程方法

    举例说明，如果要使用面向对象的思维封装小狗，可以从以下两个方面描述：

    1. 从特征描述，如颜色、重量、名字等；
    2. 从行为描述，如会叫，会跳，会睡觉等；

    通过以上两个方面描述小狗的属性和行为，将一系列数据和函数封装到一个类中，在需要使用的时候调用不同的属性或方法，就是面向对象编程的过程。

    1.1 面向对象相关术语

    • 类：可以理解为一个模板，通过类可以创建无数个具体实例。比如前面的小狗，通过它可以创建无数有不同特征和行为的小狗（这一过程又称为实例化）
    • 对象：类不能直接使用，通过类创建的实例（又称对象）才能使用。例如飞机图纸（类）本身并不能使用，但是按照图纸创建出的飞机（对象）就可以使用。
    • 属性：类中所有变量称为属性，如小狗的颜色、重量、名字等特征。
    • 方法：类中所有函数称为方法。如小狗会叫、会跳、会睡觉的函数。调用这些方法，可以使小狗做出不同行为。

    02. 类的定义

    类（英文名 class），是具有相同特性（属性）和行为（方法）的对象（实例）的抽象模板。

    类与对象的关系就如模具和铸件的关系，类的实例化的结果就是对象，而对象的抽象就是类，类描述了一组有相同特性（属性）和相同行为（方法）的对象。

    2.1 class 定义类

    在python中，定义类使用class 关键字实现，基本语法如下:

    class 类名:
    多个（>=0）类属性
    多个（>=0）类方法
    • 类属性：在类中，所有函数之外定义的变量，成为类属性或类变量
    • 类方法：在类中，定义的所有函数，成为类方法
    💡 类名应使用大驼峰命名方法，即首字母大写，如`Student` 。如果有多个单词，每个单词的首字母需要大写，如`MyFirstDemo` 
    class MyFirstDemo:
    """类的注释文档"""
    age = 18

    def __init__(self, name):
        self.name = name

    def run(self):
        print(f"{self.name}跑起来了")

    2.2 类的实例化

    类的实例化是将定义好的类定义给一个对象，通过对象.类方法或对象.类属性的方式可以调用类中定义的方法和变量。

    • 在实例化过程中，如果构造的类没有手动添加__init__() 函数，又或者添加的__init__() 函数中除了self 没有其他的参数，则创建对象时的参数可以省略不写

    如下代码创建了名为MyFirstDemo 的类，并对齐进行了实例化：

    class MyFirstDemo:
    """类的注释文档"""
    age = 18

    def __init__(self, name):
        self.name = name

    def run(self):
        print(f"{self.name}跑起来了")

dog = MyFirstDemo(name="旺财")

    在实例化过程中，传参可以使用函数的位置传参、关键字传参。self 参数python会自动传递，无需传递实参。

    2.3 类对象的使用

    定义的类只有进行实例化，也就是使用该类创建对象之后，才能使用。实例化后的类对象可以执行以下操作：

    • 访问或修改类对象具有的实例变量，甚至可以添加新的实例变量或删除已有的实例变量
    • 调用类对象的方法，包括调用现有的方法，以及给类对象动态添加方法

    以下代码演示将MyFirstDemo 实例化后，访问类对象的实例变量

    class MyFirstDemo:
    """类的注释文档"""
    age = 18

    def __init__(self, name):
        self.name = name

    def run(self):
        print(f"{self.name}跑起来了")

dog = MyFirstDemo(name="旺财")
print(dog.age)
>>>
18

    将类实例化后，可以使用实例对象.变量的方式修改实例对象的值

    dog.age = 21
print(dog.age)
>>>
21

    给类对象动态添加/删除变量

    dog.whight = "10斤"
print(dog.whight)
>>>
10斤

# 使用del关键字可以动态删除类属性
del dog.whight
print(dog.whight)
>>>
AttributeError: 'MyFirstDemo' object has no attribute 'whight'

    类对象方法的调用，如果方法内仅有self 一个参数，则不需要传参

    dog.run()
>>>
旺财跑起来了

    给类对象动态添加方法

    先定义一个函数
def info(self):
    print("这是新增的info方法", self)

#使用info对dog对象的foo方法赋值（动态绑定方法）
dog.foo = info
# python不会自动将调用者绑定第一参数，即使传递self参数也没用。
dog.foo()
>>>
TypeError: info() missing 1 required positional argument: 'self'

#调用时需要传递类对self进行绑定
dog.foo(MyFirstDemo)
>>>
这是新增的info方法 <class '__main__.MyFirstDemo'>

    03. 内置方法和属性

    3.1 self 参数

    💡 同一个类可以产生多个对象，当某个对象调用类方法时，该对象会把自身的引用作为第一个参数自动传给该方法，Python 会自动绑定类方法的第一个参数指向调用该方法的对象。如此，Python解释器就能知道到底要操作哪个对象的方法了。无论是类中的构造函数还是普通的类方法，实际调用它们的谁，则第一个参数 self 就代表谁。

    如果把类比作造房子的图纸，那么类实例化后的对象是真正可以住的房子。根据一张图纸（类），可以设计出成千上万的房子（类对象），每个房子长相都是类似的（都有相同的类变量和类方法），但它们都有各自的主人，那么如何对它们进行区分呢？

    通过 self 参数，它就相当于每个房子的门钥匙，可以保证每个房子的主人仅能进入自己的房子（每个类对象只能调用自己的类变量和类方法）。

    class Person:
    def __init__(self):
        print("正在执行构造方法")

    # 定义一个study()实例方法
    def study(self):
        print(self, "正在学习python")

zhangsan = Person()
zhangsan.study()
print("--------")
lisi = Person()
lisi.study()

>>>
正在执行构造方法
<__main__.Person object at 0x7f806dfb3358> 正在学习python
--------
正在执行构造方法
<__main__.Person object at 0x7f806dfb33c8> 正在学习python

    以上的代码中，study中的self 代表该方法的调用者，即谁调用该方法，那么self 就代表谁。

    3.2 __init__ 初始化方法

    在创建类时，可以手动添加__init__() 方法，该方法是一个特殊的类实例方法，成为构造方法（或构造函数或初始化方法）

    • 构造方法用于创建对象时使用，每当创建一个类的实例对象时，解释器会自动调用它。初始化方法语法格式如下：
    def __init__(self,*args):
    代码块

    __init__ 方法可以包含多个参数，但必须包含一个名为self 的参数，且为第一个参数。

    class Person:
    def __init__(self):
        print("正在执行构造方法")

    # 定义一个study()实例方法
    def study(self):
        print(self, "正在学习python")

zhangsan = Person()
>>>
正在执行构造方法

    构造函数是在创建类对象时就开始运行

    3.3 __del__ 方法

    • 在python中，以类型() 创建对象时，为对象分配完空间后，自动调用__init__() 方法
    • 当一个对象被从内存中销毁前，会自动调用__del__() 方法

    **__del__ 方法的应用场景**

    • __init__ 改造初始化方法，可以让创建对象更加灵活
    • __del__ 如果希望在对象被销毁前，再做一些事情，可以考虑一下 __del__ 方法

    类对象的生命周期

    • 一个对象从调用 类名() 创建，生命周期开始
    • 一个对象的 __del__ 方法一旦被调用，生命周期结束
    • 在对象的生命周期内，可以访问对象属性，或者让对象调用方法
    class Person:
    age = 18
    def __init__(self, name):
        self.name = name
        print("%s开始运行" % self.name)

    def __del__(self):
        print("%s运行结束"% self.name)

zhangsan = Person("小明")
print(zhangsan.name)
print(zhangsan.age)
>>>
小明开始运行
小明
18
小明运行结束

    3.4 __str__ 方法

    • 在 Python 中，使用 print 输出 对象变量，默认情况下，会输出这个变量 引用的对象 是 由哪一个类创建的对象，以及 在内存中的地址（十六进制表示）
    • 如果在开发中，希望使用 print 输出 对象变量 时，能够打印 自定义的内容，就可以利用 __str__ 这个内置方法了

    注意：str 方法必须返回一个字符串 

    class Person:
    def __init__(self, name):
        self.name = name
        print("%s开始运行" % self.name)

    def __del__(self):
        print("%s运行结束" % self.name)

    def __str__(self):
        return "大家好，我叫%s" % self.name

xiaoming = Person("小明")
print(xiaoming)
>>>
小明开始运行
大家好，我叫小明
小明运行结束
