    多态

    01. 多态的定义

    不同的子对象，调用相同的父类方法，产生不同的结果执行；

    • 继承：多态一定是发生在子类和父类之间；
    • 重写：子类重写了父类方法
    class A:
    def say(self):
        print("这是A类下的say方法")

class B(A):
    def say(self):
        print("这是B类下的say方法")

demo_a = A()
demo_a.say()

demo_b = B()
demo_b.say()
>>>
这是A类下的say方法
这是B类下的say方法

    02. 多态演练

    class Dog:
    def __init__(self, name):
        self.name = name

    def game(self):
        print("{}蹦蹦跳跳的玩耍...".format(self.name))

class XiaoTianDog(Dog):
    def game(self):
        print("{}在天上玩耍".format(self.name))

class Person:
    def __init__(self, name):
        self.name = name

    def game_with_dog(self, dog):  # dog参数接受dog或XiaoTiandog类对象
        print("{}和{}快乐的玩耍".format(self.name, dog.name))
        dog.game()  # 执行dog对象的实例方法

# 创建一条旺财的狗对象
wangcai = Dog("旺财")
# 定义小明的人类对象
xiaoming = Person("小明")
# 调用Person类类game_with_dog的方法
xiaoming.game_with_dog(wangcai)  # 传入旺财的对象
>>>
小明和旺财快乐的玩耍
旺财蹦蹦跳跳的玩耍...
