    01. 语法

    Python 条件语句是通过一条或多条语句的执行结果（True 或者 False）来决定执行的代码块。

    • if判断条件结束以： 结束;
    • 内部语句4个空格缩进
    age = 18
if age >= 18:
    print("可以去网吧了")
elif age >= 12:
    print("可以喝酒了")
else:
    print("还是乖乖的听妈妈的话吧")

    02. if中常用的操作符

    操作符 描述
    < 小于
    <= 小于或等于
    > 大于
    >= 大于或等于
    == 等于，比较两个值是否相等
    != 不等于

    03. if接变量名：

    • if 语句接变量名，如果变量非空。则运行条件代码。
    • 除了”、”“、0、()、[]、{}、None为False之外，其他的都是True
    age = 18
a = None
if age:
    print("我今年{}岁了哦".format(age))
if a:
    print("这行不会显示")

    04. if嵌套

    在嵌套 if 语句中，可以把 if...elif...else 结构放在另外一个 if...elif...else 结构中。

    num=int(input("输入一个数字："))
if num%2==0:
    if num%3==0:
        print ("你输入的数字可以整除 2 和 3")
    else:
        print ("你输入的数字可以整除 2，但不能整除 3")
else:
    if num%3==0:
        print ("你输入的数字可以整除 3，但不能整除 2")
    else:
        print  ("你输入的数字不能整除 2 和 3")
