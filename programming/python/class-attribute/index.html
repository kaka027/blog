    类变量

    无论是类属性还是类方法，都无法像普通变量或者函数那样，在类的外部直接使用它们。我们可以将类看做一个独立的空间，则类属性其实就是在类体中定义的变量，类方法是在类体中定义的函数。

    在类体中，根据变量定义的位置不同，以及定义的方式不同，类属性又可细分为以下 3 种类型：

    • 类属性(类变量)：类体中，所有函数之外，定义的变量
    • 实例属性(实例变量)：类体中，所有函数内部，以self.变量名 的方式定义的变量
    • 局部变量：类体中，所有函数内部，以变量名=变量值 的方式定义的变量。

    01. 类属性(类变量)

    类变量指的是在类中，但在各个类方法外定义的变量，举个例子：

    class Demo:
    url = "www.baidu.com"
    title = "百度"

    def say_hello(self,name):
        print("大家好，我叫{}".format(name))

    上面代码中，url 和title 就属于类变量。

    类变量的特点是，所以类的实例化对象都同时共享类变量，也就是说，类变量在所有实例化对象中是作为公用资源存在的。

    类方法和变量调用方式有两种：

    • 直接通过类名调用
    • 通过类的实例化对象调用

    1.1 通过类名调用类变量

    # 使用类名直接调用
print(Demo.url)
print(Demo.title)
# 修改类名不仅可以直接调用，还可以修改变量的值
Demo.url = "www.qq.com"
Demo.title = "腾讯"
print(Demo.url)
print(Demo.title)
>>>
www.baidu.com
百度
www.qq.com
腾讯

    因为类变量为所有实例化对象共有，通过类名修改类变量的值，会影响所有的实例化对象：

    print("修改前的类变量")
test1 = Demo()
print(test1.url)
print(test1.title)
test2 = Demo()
print(test2.url)
print(test2.title)
print("修改后，各类对象中类变量的值")
Demo.url = "www.qq.com"
Demo.title = "腾讯"
print(test1.url)
print(test1.title)
print(test2.url)
print(test2.title)
>>>
修改前的类变量
www.baidu.com
百度
www.baidu.com
百度
修改后，各类对象中类变量的值
www.qq.com
腾讯
www.qq.com
腾讯

    1.2 通过类的实例化对象调用类变量

    test1 = Demo()
print(test1.url)
print(test1.title)
>>>
www.baidu.com
百度

    通过类对象是无法修改类变量的。通过类对象对类变量赋值，其本质将不再是修改类变量的值，而是在给该对象定义新的实例变量

    但是，可以动态的为类的实例化对象添加类变量

    test1 = Demo()
test1.slogan = "百度一下，你就知道"
print(test1.slogan)
>>>
百度一下，你就知道

    02. 实例变量(实例属性)

    实例变量指的是在任意类方法内部，以“self.变量名”的方式定义的变量，其特点是只作用于调用方法的对象。实例变量只能通过对象名访问，不能通过类名访问。

    class Demo:
    def __init__(self):
        self.name = "小明"
        self.age = 18

    def say(self):
        self.weight = 120

    在Demo类中，name 、age、weight 都是实例变量。由于__Init__ 是构造函数，会在创建类的实例化对象时自动调用，而say 方法需要手动调用。因此Demo的实例化对象都会包含name和age实例变量，而只有调用了say方法，才包含weight变量

    test1 = Demo()
print(test1.name)
print(test1.age)
# 由于未调用say方法， 所以下面的这句会报错
# print(test1.weight)

test2 =Demo()
test2.say()
print(test2.weight)
>>>
小明
18
120

    2.1 使用实例化对象无法修改类变量的原因

    使用类对象，可以访问类变量，但无法修改类变量的值。这是因为，通过类对象修改类变量的值，不是在给类变量赋值，本质上是再定义新的实例变量。

    test1 = Demo()
# 类对象访问类变量
print(test1.url)
# 类对象的方式尝试修改类变量的值
test1.url = "www.qq.com"
# 再次访问，查看修改后的结果
print(test1.url)
# 查看类变量的值
print(Demo.url)
>>>
www.baidu.com
www.qq.com
www.baidu.com

    以上可以看出，通过类对象是无法修改类变量的值的，本质是在给类变量test1定义新的实例变量。

    在类中，类变量和实例变量可以同名，但是在使用类对象的方式调用同一个变量名时，python会优先选择实例变量，所以在代码编写过程中，不推荐使用类对象的方式调用类变量。

    2.2 类对象方式修改实例变量的值

    和类对象不同，通过某个对象修改实例变量的值，不会影响其他类的实例化对象，更不会影响同名的类变量：

    class Demo:
    name = "杰森斯坦森"  # 类变量
    weight = 180  # 类变量

    def __init__(self):
        self.name = "xiaomei"  # 实例变量
        self.weight = 96  # 实例变量

    # 定义一个say方法
    def say(self):
        self.type = "学生"  # 实例变量

test1 = Demo()
# 修改test1对象的实例变量
test1.name = "yaya"
test1.weight = 80
print(test1.name)
print(test1.weight)

# 定义另一个实例化对象
test2 = Demo()
print(test2.name)
print(test2.weight)

# 查看类变量的值
print(Demo.name)
print(Demo.weight)

>>>
yaya
80
xiaomei
96
杰森斯坦森
180

    2.3 类对象添加新的实例变量

    python只支持为特定的对象添加实例变量。例如在前面代码的基础上，添加ismarriage 实例变量：

    test1.ismarriage = "未婚"
print(test1.ismarriage)
print(test2.ismarriage)
>>>
未婚
AttributeError: 'Demo' object has no attribute 'ismarriage'

    03. 局部变量

    在类方法内，直接以变量名=值 的方式定义的变量，是局部变量：

    class Demo:
    # 定义一个count方法
    def count(self, money):
        sale = 0.8 * money
        print("优惠后的价格为{}".format(sale))

xigua = Demo()
xigua.count(10)
>>>
优惠后的价格为8.0

    通常情况下，局部变量是为了类方法的功能实现，不能直接通过类对象调用使用。局部变量只能用于所在函数中，函数执行完成后，局部变量也会被销毁
