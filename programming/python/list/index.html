    列表

    01. 列表的定义

    • List（列表） 是 Python 中使用 最频繁 的数据类型；
    • 专门用于存储 一串 信息
    • 列表用 [] 定义，数据 之间使用 , 分隔
    • 列表的 索引 从 0 开始。

    注意：从列表中取值时，如果 超出索引范围，程序会报错 

    list1 = [1, 2, 3, 4, 5]
list2 = ["xian", "hangzhou", "beijing", "shanghai", "tianjing"]
list3 = ["小明", 18， 187.5]

    02. 列表的取值和嵌套

    2.1 列表取值

    • 列表的索引从左到右从0开始，从右到左从-1 开始
    • 通过列表[索引]的方式可以取值、截取的操作

    以下演示列表取值和截取的实例：

    name_list = ["小明", "小李", "小张"]
# 正向索引取值
print(name_list[0])
>>>
小明

# 反向索引取值
print(name_list[-1])
>>>
小张

# 截取 -切片范围为左闭右开，截取返回一个列表
print(name_list[:2])
['小明', '小李']

    2.2 列表嵌套

    • 使用嵌套列表在列表中创建其他列表，以下实例：
    a = [1, 2, 3]
b = ["a","b","c"]
c = ["小明","小李","小张"]
x = [a, b, c]
print(x)
>>>
[[1, 2, 3], ['a', 'b', 'c'], ['小明', '小李', '小张']]

# 嵌套列表的索引是子列表
print(x[2])
>>>
['小明', '小李', '小张']

# 取子列表内的值
print(x[2][0])
>>>
小明

# 嵌套列表的切片
print(x[0:-1])
>>>

[[1, 2, 3], ['a', 'b', 'c']]
print(x[2][:-1])
>>>
['小明', '小李']

    2.3 列表复制

    • 如果需要对列表进行复制，复制后的列表和原列表相互独立，可使用切片的方式；
    • 如果想达到复制的目的，不能使用引用直接赋值，引用方法会将两个变量指向同一个列表内存地址，本质上还是同一个列表，不能相互独立
    my_fruits = ["banner", "apple", "orange"]
# 使用切片复制列表
friend_fruits = my_fruits[:]
print(my_fruits)
print(friend_fruits)
>>>
['banner', 'apple', 'orange']
['banner', 'apple', 'orange']

# 验证列表副本和原列表相互独立
my_fruits.append("pears")
friend_fruits.append("Pineapple")
print("增加后my_fruits是",my_fruits)
print("增加后friend_fruits是",friend_fruits)
>>>
增加后my_fruits是 ['banner', 'apple', 'orange', 'pears']
增加后friend_fruits是 ['banner', 'apple', 'orange', 'Pineapple']

# 引用方法赋值，这样是不可行的
my_foods = ["cake", "rice", "Cucumber"]
friend_foods = my_foods
print(my_foods)
print(friend_foods)
>>>
['cake', 'rice', 'Cucumber']
['cake', 'rice', 'Cucumber']

# 验证列表副本和原列表是否相互独立
my_foods.append("Vegetables")
friend_foods.append("Noodles")
print("append后my_foods:",my_foods)
print("append后friend_foods:",friend_foods)
>>>
append后my_foods: ['cake', 'rice', 'Cucumber', 'Vegetables', 'Noodles']
append后friend_foods: ['cake', 'rice', 'Cucumber', 'Vegetables', 'Noodles']

    03. 列表内置方法

    • 在ipython3定义一个列表，例如name_list=['小明', '小李', '小张']
    • 键入name_list. 会提示列表能够使用的方法如下：
    In [1]: name_list.
name_list.append   name_list.count    name_list.insert   name_list.reverse
name_list.clear    name_list.extend   name_list.pop      name_list.sort
name_list.copy     name_list.index    name_list.remove

    3.1 增加 -3

    方法 说明
    列表.insert(索引, 数据) 在指定位置插入数据
    列表.append(数据) 在末尾追加数据
    列表.extend(列表2) 将列表2 的数据追加到列表 
    # list.insert() -没有返回值，但会在列表指定位置插入对象
name_list = ["小明", "小李", "小张"]
name_list.insert(1, "小花")
print(name_list)
>>>
['小明', '小花', '小李', '小张']

# list.append()
name_list = ["小明", "小李", "小张"]
name_list.append("小花")
print(name_list)
>>>
['小明', '小李', '小张', '小花']
    • list.extend方法在列表末尾一次性追加另一个序列中的多个值；
    • 参数seq - -元素列表，可以是列表、元组、集合、字典，若为字典,则仅会将键(key)作为元素依次添加至原列表的末尾。
    name_list = ["小明", "小李", "小张"]
a_list = [1, 2, 3]
name_list.extend(a_list)
print(name_list)

>>>
['小明', '小李', '小张', 1, 2, 3]

    不同数据类型的追加实例：

    name_list = ["小明", "小李", "小张"]
tup_list = (1, 2, 3)
dict_list = {"name": "小明", "age": 18, "weight": 65}
set_list = {"chinese", "Japanese"}

# 元组追加
name_list.extend(tup_list)
print(name_list)
>>>
['小明', '小李', '小张', 1, 2, 3]

# 字典追加
name_list.extend(dict_list)
print(name_list)
>>>
['小明', '小李', '小张', 'name', 'age', 'weight']

# 集合追加
name_list.extend(set_list)
print(name_list)
>>>
['小明', '小李', '小张', 'chinese', 'Japanese']

    3.2 修改

    • list[索引]=“new”
    name_list = ["小明", "小李", "小张"]
print("第二个姓名是{}".format(name_list[1]))
name_list[1] = "张大炮"
print("修改后第二个姓名是{}".format(name_list[1]))
>>>
第二个姓名是小李
修改后第二个姓名是张大炮

    3.3 删除 -5

    方法 说明
    del 列表[索引] 删除指定索引的数据
    列表.remove[数据] 删除第一个出现的指定数据
    列表.pop 删除末尾数据
    列表.pop(索引) 删除指定索引数据
    列表.clear 清空列表

    1.del 列表[索引] - del删除是在内存中删除，后续无法再使用

    name_list = ["小明", "小李", "小张"]
del name_list[0]
print(name_list)
>>>
['小李', '小张']

    2.list.remove[数据] - 参数-obj：列表中需要移除的对象数据，非下标

    name_list = ["小明", "小李", "小张", "小明"]
name_list.remove("小明")
print(name_list)
>>>
['小李', '小张', '小明']

    3.list.pop(index=-1)

    name_list = ["小明", "小李", "小张", "小明"]
print("现在列表是",name_list)
name_list.pop()
print("使用pop后，列表为",name_list)
>>>
现在列表是 ['小明', '小李', '小张', '小明']
使用pop后，列表为 ['小明', '小李', '小张']

    4.list.pop(index) - 使用pop删除后，后续还能使用删除的数据

    name_list = ["小明", "小李", "小张", "小明"]
print("现在列表是",name_list)
name_list.pop(-1)
print("使用pop后，列表为",name_list)
>>>
现在列表是 ['小明', '小李', '小张', '小明']
使用pop后，列表为 ['小明', '小李', '小张']

pop_list = name_list.pop(-1)
print("刚使用pop删除的数据是{}".format(pop_list))
print("现在列表数据是", name_list)
>>>
刚使用pop删除的数据是小明
现在列表数据是 ['小明', '小李', '小张']

    5.list.clear()

    name_list = ["小明", "小李", "小张", "小明"]
print("现在列表是",name_list)
name_list.clear()
print("使用clear后，列表为",name_list)
>>>
现在列表是 ['小明', '小李', '小张', '小明']
使用clear后，列表为 []

    3.5 排序

    方法 说明
    列表.sort() 升序排序
    列表.sort(reverse=True) 降序排序
    列表.sorted(reverse) 临时排序
    列表.reverse() 逆序、反转

    1.list.sort()

    • sort方法对于列表永久性排序，无法恢复原样
    a = [5, 3, 6, 1]
a.sort()
print(a)
>>>
[1, 3, 5, 6]

a = [5, 3, 6, 1]
a.sort(reverse=True)
print(a)
>>>
[6, 5, 3, 1]

    2.sorted(可迭代对象, reverse)

    • sort仅可以用在列表上，sorted可以使用在所有可迭代数据类型上；
    • sorted返回的是一个新的list，不影响原来list的排序
    # reverse默认为False，升序
a = [5, 3, 6, 1
new_a = sorted(a)
print(new_a)
print(a)
>>>
[1, 3, 5, 6]
[5, 3, 6, 1]

# 降序
a = [5, 3, 6, 1]
new_a = sorted(a,reverse=True)
print(new_a)
print(a)
>>>
[6, 5, 3, 1]
[5, 3, 6, 1]

    3.list.reverse()

    city = ["hangzhou", "beijing", "shenzhen"]
city.sort()
print("升序排列后",city)
city.reverse()
print("逆序为", city)
>>>
升序排列后 ['beijing', 'hangzhou', 'shenzhen']
逆序为 ['shenzhen', 'hangzhou', 'beijing']

    列表脚本操作符

    • 列表对 + 和 * 的操作符与字符串相似。+ 号用于组合列表，* 号用于重复列表。
    Python 表达式 结果 描述
    len([1, 2, 3]) 3 长度
    [1, 2, 3] + [4, 5, 6] [1, 2, 3, 4, 5, 6] 组合
    ['Hi!'] * 4 ['Hi!', 'Hi!', 'Hi!', 'Hi!'] 重复
    3 in [1, 2, 3] True 元素是否存在于列表中
    for x in [1, 2, 3]:
    print(x, end=" ")    		 1 2 3 迭代
