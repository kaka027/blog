    01. 元组的定义

    • 元组与列表类似，不同之处在于元素不能修改；
    • 元组使用()定义，使用,分割；
    • 元组的索引从0开始
    tup = ("小明", "小张", "小李")
print(tup)
print(type(tup))
>>>
('小明', '小张', '小李')

>>>
<class 'tuple'>

# 不加括号也可以定义元组
tup3 = "a", 1.89, 20, "good"
print(type(tup3))
>>>
<class 'tuple'>

    1.1 定义空元组

    tup = ()

    1.2 元组中只包含一个元素

    • 元组中只包含一个元素时，需要在元素后面加,
    tup = (50)
print(type(tup))
>>>
<class 'int'>

tup2 = (50,)
print(type(tup2))
>>>
<class 'tuple'>

    02. 元组的取值和截取

    • 因为元组也是一个序列，所以我们可以访问元组中的指定位置的元素，也可以截取索引中的一段元素

    2.1 元组的访问和截取

    • 元组可以使用下标索引来访问元组中的值：
    tup3 = ("小明", 18, 1.75)
print(tup3[0])
print(tup3[:])
print("我的名字叫{},我的年龄是{}岁".format(*tup3))
>>>
小明
('小明', 18, 1.75)
我的名字叫小明,我的年龄是18岁

    2.2 元祖修改

    • 元组中的元素值是不允许修改的，但可以对元组进行连接组合，如下实例:
    tup1 = (1, 2, 3)
tup2 = ("a","b", "c")

# 通过下标修改是非法的
# tup1[0] = 2
# TypeError: 'tuple' object does not support item assignment

# 创建一个新的元组
tup3 = tup1 + tup2
print(tup3)
>>>
(1, 2, 3, 'a', 'b', 'c')

tup1 += tup2
print(tup1)
>>>
(1, 2, 3, 'a', 'b', 'c')

    2.3 元组删除

    • 元组中的元素值是不允许删除的，但我们可以使用del语句来删除整个元组。
    • del关键字本质上是将元组从内存中删除，所以后续无法再调用
    tuple = ("hello", "python", "world")
print(tuple)
>>>
('hello', 'python', 'world')

del tuple
print(tuple)
>>>
NameError: name 'tup' is not defined

    03. 元组的运算

    • 与字符串一样，元组之间可以使用 + 号和 * 号进行运算。这就意味着他们可以组合和复制，运算后会生成一个新的元组。
    Python 表达式 结果 描述
    len((1, 2, 3)) 3 计算元素个数
    (1, 2, 3) + (4, 5, 6) (1, 2, 3, 4, 5, 6) 连接
    ('Hi!',) * 4 ('Hi!', 'Hi!', 'Hi!', 'Hi!') 复制
    3 in (1, 2, 3) True 元素是否存在
    for x in (1, 2, 3): print (x,) 1 2 3 迭代 
    tup = ("a", "b", "c")
print(len(tup))
>>>
3

tup = ("a", "b", "c")
tup1 = (1, 2, 3)
tup += tup1
print(tup)
>>>
('a', 'b', 'c', 1, 2, 3)

tup1 = (1, 2, 3)
print(tup1 * 3)
>>>
(1, 2, 3, 1, 2, 3, 1, 2, 3)

tup1 = (1, 2, 3)
print(3 in tup1)
>>>
True

tup1 = (1, 2, 3)
for x in tup1:
    print(x)
>>>
1
2
3

    04. 元组的内置函数

    • Python元组包含了以下内置函数
    序号 方法及描述 实例
    1 len(tuple)计算元组元素个数。 >>> tuple1 = ('Google', 'Runoob', 'Taobao')
    >>> len(tuple1)
    3
    2 max(tuple)返回元组中元素最大值。 >>> tuple2 = ('5', '4', '8')
    >>> max(tuple2)
    '8'
    3 min(tuple)返回元组中元素最小值。 >>> tuple2 = ('5', '4', '8')
    >>> min(tuple2)
    '4'
    4 tuple(iterable)将可迭代系列转换为元组。 >>> list1= ['Google', 'Taobao', 'Runoob', 'Baidu']
    >>> tuple1=tuple(list1)
    >>> tuple1
    ('Google', 'Taobao', 'Runoob', 'Baidu')

    05. 元组的方法

    • Python 有两个可以在元组上使用的内建方法:
    方法 描述
    count() 返回元组中指定值出现的次数。
    index() 在元组中搜索指定的值并返回它第一次被找到的位置。 
    tup1 = (1, 2, 3, 1, 2, 4, 5)
print(tup1.count(1))
>>>
2

tup1 = (1, 2, 3, 1, 2, 4, 5)
print(tup1.index(1))
>>>
0
