    类方法

    01. 写法的差异

    类的方法有以下三种：

    • 静态方法：在类中，以@staticmethod 装饰的函数；
    • 类方法：在类中，以@classmethod 装饰的函数；
    • 实例方法：没有任何装饰器的函数。@staticmethod

    以下代码中，run 是实例方法，eat 是静态方法，jump 是类方法。

    class Animal:
    def __init__(self, name):
        self.name = name

    def run(self):
        print(f"{self.name}跑起来啦")

    @staticmethod
    def eat():
        print("正在吃饭...")

    @classmethod
    def jump(cls, name):
        print(f"{name}跳起来啦")

    02. 实例方法

    实例方法最大的特点就是，它最少也要包含一个 self 参数，用于绑定调用此方法的实例对象（Python 会自动完成绑定）。

    实例方法有两种调用方式，以实例对象调用和类直接调用。

    2.1 类的实例化对象调用-绑定方法

    dog = Animal(name="旺财")
dog.run()
>>>
旺财跑起来啦

    2.2 通过类直接调用-非绑定方法

    使用类名直接调用实例方法， 需要给self手动传参，否则会报错

    dog = Animal(name="旺财")
Animal.run()
>>>
TypeError: run() missing 1 required positional argument: 'self'

dog = Animal(name="旺财")
Animal.run(dog)
>>>
旺财跑起来啦

    03. 类方法

    类方法是在类中，由@classmethod 装饰的方法。需要至少包含一个参数，通过以cls 命名，为class 的简写，代表类本身。

    类方法的可以以实例化对象和类调用，两种调用方式都不需要为cls 传参。一般不推荐使用实例化对象的方式调用类方法

    dog = Animal(name="旺财")
dog.jump("小黑")
Animal.jump("小黑")
>>>
小黑跳起来啦
小黑跳起来啦

    04. 静态方法

    静态方法是在类中，由@staticmethod 装饰的方法。静态方法本质是一个函数，没有固定参数。

    静态方法的可以使用实例化对象和类直接调用两种方式。

    dog = Animal(name="旺财")
dog.eat()
Animal.eat()
>>>
正在吃饭...
正在吃饭...

    05. 方法与函数的区别

    • 普通函数和静态方法（@staticmethod）都是函数（function）。
    • 实例方法和类方法（@classmethod）都是方法（method）。

    这些结论，可以使用type 函数得到验证：

    class Animal:
    def __init__(self, name):
        self.name = name

    def run(self):
        print(f"{self.name}跑起来啦")

    @staticmethod
    def eat():
        print("正在吃饭...")

    @classmethod
    def jump(cls, name):
        print(f"{name}跳起来啦")

def demo_func():
    pass

dog = Animal(name="旺财")
print(type(demo_func))
print(type(dog.eat))
print("----")
print(type(dog.run))
print(type(dog.jump))
>>>
<class 'function'>
<class 'function'>
----
<class 'method'>
<class 'method'>
