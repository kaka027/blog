    推导式

    01. 推导式的定义

    推导式又称解析式，是python一种独有的特性。推导式是可以从一个数据序列构建另一个新的数据序列的结构体。共有四种推导式：

    • 列表推导式
    • 元组推导式
    • 字典推导式
    • 集合推导式

    02. 列表推导式

    语法格式：

    variable = [out_exp_res for out_exp in input_list if out_exp == 2]
    • out_exp_res:列表生成元素表达式，可以是有返回值的函数。
    • for out_exp in input_list : 迭代input_list 将out_exp传入out_exp_res表达式中。
    • if out_exp ==2 :根据条件过滤哪些值。

    实例：

    • 过滤列表中，字符长度大于3的元素，并将剩下的转换为大写：
    names = ['Bob', 'Tom', 'alice', 'Jerry', 'Wendy', 'Smith']
a = [name.upper() for name in names if len(name)<3]
print(a)
>>>
['ALICE', 'JERRY', 'WENDY', 'SMITH']
    • 列表推导式的循环嵌套
    a = [(x , y)for x in range(5) for y in range(4) if x != 0 and y !=0]
print(a)
>>>
[(1, 1), (1, 2), (1, 3), (2, 1), (2, 2), (2, 3), (3, 1), (3, 2), (3, 3), (4, 1), (4, 2), (4, 3)]

    03. 元组推导式

    元组推导式和列表推导式的语法一样，只是将[] 换为()

    (表达式 for 迭代变量 in 可迭代对象 [if 条件表达式] )
    • 其中，用 [] 括起来的部分，可以使用，也可以省略。

    使用元组推导式生成包含数字1-9的元组

    a = (x for x in range(1,10))
print(a)
>>>
<generator object <genexpr> at 0x7f8d8b39de60>

    从上面的执行结果可以看出，使用元组推导式生成的结果并不是一个元组，而是一个生成器对象，这一点和列表推导式是不同的。

    如果我们想要使用元组推导式获得新元组或新元组中的元素，有以下三种方式:

    1. 使用 tuple() 函数，可以直接将生成器对象转换成元组，例如：
    a = (x for x in range(1,10))
print(tuple(a))
>>>
(1, 2, 3, 4, 5, 6, 7, 8, 9)
    1. 直接使用 for 循环遍历生成器对象，可以获得各个元素，例如：
    a = (x for x in range(1,10))
for i in a:
    print(i,end="")
print(tuple(a))
>>>
123456789()
    1. 使用 __next__() 方法遍历生成器对象，也可以获得各个元素，例如：
    a = (x for x in range(3))
print(a.__next__())
print(a.__next__())
print(a.__next__())
a = tuple(a)
print("转换后的元组：",a)
>>>
0
1
2
转换后的元组： ()

    注意，无论是使用 for 循环遍历生成器对象，还是使用__next__()方法遍历生成器对象，遍历后原生成器对象将不复存在，这就是遍历后转换原生成器对象却得到空元组的原因。

    04. 字典推导式

    • 字典推导式由{} 包裹，语法和列表及元组一样；
    • 表达式中需要定义key 和value 两个变量
    listdemo = ['百度','www.baidu.com']
#将列表中各字符串值为键，各字符串的长度为值，组成键值对
newdict = {key:len(key) for key in listdemo}
print(newdict)
>>>
{'百度': 2, 'www.baidu.com': 13}
    • 交换字典中各键值对的键和值：
    xiaoming_dict = {"name": "xiaoming", "age": 18, "weight": 100}
a = {k: v for v, k in xiaoming_dict.items()}
print(a)
>>>
{'xiaoming': 'name', 18: 'age', 100: 'weight'}
    • 使用 if 表达式筛选符合条件的键值对:
    xiaoming_dict = {"name": "xiaoming", "age": 18, "weight": 100}
a = {k: v for v, k in xiaoming_dict.items() if k == "xiaoming"}
print(a)
>>>
{'xiaoming': 'name'}

    05. 集合推导式

    • 集合推导式和字典推导式都是以{} 包裹
    • 判断该推导式是字典还是集合的方式是看表达式内是单个变量还是键值对。如果是键值对则是字典推导式，反之为集合推导式
    # 由于是集合，所以保存的数据是唯一的
a = {0,1,2,1,2,3,4}
b = {s for s in a}
print(b)
>>>
{0, 1, 2, 3, 4}
    • 使用if作为条件的集合推导式：
    tupledemo = (1,1,2,3,4,5,6,6)
setnew = {x**2 for x in tupledemo if x%2==0}
print(setnew)
>>>
{16, 4, 36}
    • 使用集合推导式变量字典的key，保存为集合
    dictdemo = {'1':1,'2':2,'3':3}
setnew = {x for x in dictdemo.keys()}
print(setnew)
>>>
{"2","1","3"}
