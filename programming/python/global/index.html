    变量作用域

    01. 局部变量

    • 在函数内部定义的变量，作用域也仅是函数内，函数执行时分配内存空间，函数执行完毕，自动回收。
    def test():
    text = "www.baidu.com"
    print("函数内部",text)

test()
print("函数外部",text)
>>>
函数内部 www.baidu.com
NameError: name 'text' is not defined

    函数的参数也是局部变量

    def people(name,age):
    print("内部name",name)
    print("内部age",age)

people("小明",18)
print("外部",name)
>>>
内部name 小明
内部age 18
NameError: name 'name' is not defined

    02.全局变量

    全局变量-Global Variable

    • 和局部变量不同，全局变量的默认作用域是整个程序，即全局变量既可以在各个函数的外部使用，也可以在各函数内部使用。

    定义全局变量的方式有两种：

    • 1.在函数外部定义的变量一定是全局变量
    add = "www.baidu.com"
def text():
    print("函数体内访问：",add)
text()
print('函数体外访问：',add)
>>>
函数体内访问： www.baidu.com
函数体外访问： www.baidu.com
      1. 在函数内部使用global 关键字定义变量，也是全局变量：
    def text():
    global url
    url = "www.baidu.com"
    print("函数内部",url)

text()
print("函数外部",url)
>>>
函数内部 www.baidu.com
函数外部 www.baidu.com

    03. 获取指定作用域中的变量

    3.1 globals()函数

    • globals() 函数为 Python 的内置函数，它可以返回一个包含全局范围内所有变量的字典，该字典中的每个键值对，键为变量名，值为该变量的值。
    #全局变量
Pyname = "百度"
Pyadd = "www.baidu.com"
def text():
    #局部变量
    Shename = "腾讯"
    Sheadd= "www.qq.com"
print(globals())
>>>
'Pyname': '百度', 'Pyadd': 'www.baidu.com'
    • 通过调用 globals() 函数，可以得到一个包含所有全局变量的字典。并且，通过该字典，我们还可以访问指定变量，甚至如果需要，还可以修改它的值。例如:
    print(globals()['Pyname'])
print(globals()["Pyadd"])
globals()["Pyname"] = "baidu"
print(Pyname)
>>>
百度
www.baidu.com
baidu

    3.2 locals()函数

    locals() 函数也是 Python 内置函数之一，通过调用该函数，我们可以得到一个包含当前作用域内所有变量的字典。这里所谓的“当前作用域”指的是，在函数内部调用 locals() 函数，会获得包含所有局部变量的字典；而在全局范文内调用 locals() 函数，其功能和 globals() 函数相同

    #全局变量
Pyname = "百度"
Pyadd = "www.baidu.com"
def text():
    #局部变量
    Shename = "腾讯"
    Sheadd= "www.qq.com"
    print("函数内部的 locals:")
    print(locals())
text()
print("函数外部的 locals:")
print(locals())
>>>
函数内部的 locals:
{'Sheadd': 'www.qq.com', 'Shename': '腾讯'}
函数外部的 locals:
'Pyname': '百度', 'Pyadd': 'www.baidu.com'
    • 当使用 locals() 函数获得所有局部变量组成的字典时，可以向 globals() 函数那样，通过指定键访问对应的变量值

    3.3 vars(object)

    其功能是返回一个指定 object 对象范围内所有变量组成的字典。如果不传入object 参数，vars() 和 locals() 的作用完全相同。

    Pyname = "百度"
Pyadd = "www.baidu.com"
class Demo:
    name = "腾讯"
    add = "www.qq.com"
print("有 object：")
print(vars(Demo))
print("无 object：")
print(vars())
>>>
有 object：
{'name': '腾讯', 'add': 'www.qq.com'}
无 object：
{'Pyname': '百度', 'Pyadd': 'www.baidu.com'}
