    循环

    01. while 循环的基础使用

    • 循环的作用就是让 指定的代码 重复的执行
    • while 循环最常用的应用场景就是 让执行的代码 按照 指定的次数 重复 执行
    # 定义计数器
a = 0
# 定义记录最终结果的变量
result = 0
while a <= 100:
    result += a
    a += 1
print(result)
>>>
5050

    02. for 循环的基础使用

    2.1 for循环定义和语法

    • for 循环用于遍历可迭代对象。如字符串、元组、列表、字典、集合；
    • 逐个获取序列中的各个元素；
    # for 循环的语法格式如下：
for 迭代遍历 in 序列：
    代码块

    语法格式中，迭代变量用于存放从序列变量中读取的元素。

    2.2 遍历字符串

    url = "https://www.baidu.com"
for i in url:
    print(i,end="")
>>>
https://www.baidu.com

    实例中，迭代变量依次接受url 变量中的每个字符，并存入迭代遍历i 中。end=”” 表示不换行展示。

    2.3 遍历列表和元组

    # 遍历元组
tup = ("小明", 18, "boy")
for i in tup:
    print(i)
>>>
小明
18
boy

    # 遍历列表
list1 = ["小明", 18, "boy"]
for i in list1:
    print(i)
>>>
小明
18
boy

    2.4 遍历字典和集合

    • 遍历字典有三种方式，分别为keys() values() items()
    dict1 = {"name": "小明","age": 18,"身高": 1.75}
for i in dict1.keys():
    print(i)
>>>
name
age
身高

    遍历字典本身就是遍历字典的key。所以如果只想获取字典的key，可以直接遍历字典变量名，或者通过上述实例，使用字典的keys() 方法。

    dict1 = {"name": "小明","age": 18,"身高": 1.75}
for i in dict1.items():
    print(i)
>>>
('name', '小明')
('age', 18)
('身高', 1.75)

    items() 方法是遍历字典键值对。返回是元组。

    dict1 = {"name": "小明","age": 18,"身高": 1.75}
for i in dict1.values():
    print(i)
>>>
小明
18
1.75

    values() 方法是遍历字典的值

    03 .break 跳出循环

    通过break 语句可以使循环体在遍历所有项目之前跳出循环体，终止循环。

    a = [1, 2, 3]
for i in a:
    if i == 2:
        break
    print(i)
>>
1

    04. continue 跳过当前，进入下一次循环

    continue 语句可以是循环体在遍历中，跳过某个元素，直接进行下一次循环。比如从1数到5，当数到3时跳过。

    for i in range(6):
    if i == 3:
        continue
    print(i)
>>>
0
1
2
4
5

    05. 循环中的else语句

    循环中的else字句，在穷尽序列（for循环）或条件变为false（while循环）导致循环终止时被执行，但循环被break终止时，不会执行。

    for n in range(2, 10):
    for x in range(2, n):
        if n % x == 0:
            print(n, '等于', x, '*', n//x)
            break
    else:
        # 循环中没有找到元素
        print(n, ' 是质数')
>>>
2  是质数
3  是质数
4 等于 2 * 2
5  是质数
6 等于 2 * 3
7  是质数
8 等于 2 * 4
9 等于 3 * 3

    06. 标志作为循环条件

    6.1 标志

    • 将变量直接作为循环条件
    • 当变量条件为False时退出循环。以保持程序循环持续进行。
    a = True
while a:
    ms_a = input("请输入信息，输入quit退出：")
    if ms_a == "quit":
        break
    else:
        print(ms_a)
print("循环结束")
>>>
ms_a = "hello boy"
hello boy
ms_a = "quit"
循环结束

    6.2 while True

    使用while True 将使程序持续循环，直到遇到break 才会终止

    while True:
    ms = input("请输入信息，输入0退出：")
    if ms == "0":
        break
    else:
        print(ms)
print("循环结束")
>>>
ms = "hello"
hello
ms = "0"
循环结束

    07. range函数

    7.1 range 函数语法

    • range函数生成一系列数字。默认从0开始
    • 如果指定范围，生成的数字是左闭右开的，如要range(1,4)。不包含4
    for i in range(3):
    print(i,end=",")
>>>
0,1,2

for i in range(1,4):
    print(i,end=",")
>>>
1,2,3

    7.2 range切片

    可以指定步长，设置指定数字

    for i in range(1,10,2):
    print(i,end=",")
>>>
1,3,5,7,9,

    7.3 range列表

    • range函数返回不是列表。如果要将取值转换为列表，可以使用list() 函数：
    list1 = list(range(2,11,2))
print(list1)
>>>
[2, 4, 6, 8, 10]

    08. 循环嵌套

    当 2 个（甚至多个）循环结构相互嵌套时，位于外层的循环结构常简称为外层循环或外循环，位于内层的循环结构常简称为内层循环或内循环。

    循环嵌套结构的代码，Python 解释器执行的流程为：

    1. 当外层循环条件为 True 时，则执行外层循环结构中的循环体；
    2. 外层循环体中包含了普通程序和内循环，当内层循环的循环条件为 True 时会执行此循环中的循环体，直到内层循环条件为 False，跳出内循环；
    3. 如果此时外层循环的条件仍为 True，则返回第 2 步，继续执行外层循环体，直到外层循环的循环条件为 False；
    4. 当内层循环的循环条件为 False，且外层循环的循环条件也为 False，则整个嵌套循环才算执行完毕。
    adj = ["red", "big", "tasty"]
fruits = ["apple", "banana", "cherry"]

for x in adj:
  for y in fruits:
    print(x, y)

    # 以上示例返回结果为：
red apple
red banana
red cherry
big apple
big banana
big cherry
tasty apple
tasty banana
tasty cherry
