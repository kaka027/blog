    闭包和匿名

    01. 闭包函数的定义

    • 闭包函数：内部函数对外部函数 作用域 里的变量的引用
    • 闭包函数的外部函数的返回值是内部函数。
    • 闭包内的内部函数私有化了外部函数的变量，完成了数据的封装。

    以下演示，使用闭包函数某个数值的平方和立方：

    def func(n):
    def func1(m):
        return m ** n

    return func1

pingfang = func(2)  # 计算一个数的平方
lifang = func(3)  # 计算一个数的立方

# 计算2的平方和立方
print(pingfang(2))
print(lifang(2))
>>>
4
8

    以上函数中，在执行完func(2) 和func(3) 后，外部函数的参数n 会和return 部分的内部参数func1 一起赋值给变量pingfang 和lifang 。这样之后再调用pingfang和lifang 时，程序会正常执行，不会报错n 未被定义。 外部函数运行结束后，由变量接收了外部函数内的参数以及内部函数。所以外部函数的内存释放不会影响到其参数的释放。

    02. 闭包的__closure__属性

    闭包比普通的函数多了一个 closure 属性，该属性记录着自由变量的地址。当闭包被调用时，系统就会根据该地址找到对应的自由变量，完成整体的函数调用。

    以 func()为例，当其被调用时，可以通过 __**closure__** 属性获取自由变量（也就是程序中的 n 参数）存储的地址，例如：

    def func(n):
    def func1(m):
        return m ** n

    return func1

pingfang = func(2)  # 计算一个数的平方
lifang = func(3)  # 计算一个数的立方

print(pingfang.__closure__)
>>>
(<cell at 0x000001C9613BAA38: int object at 0x00007FFC87168130>,)

    可以看到，显示的内容是一个 int 整数类型，这就是 func() 中自由变量 n 的初始值。还可以看到，__**closure__** 属性的类型是一个元组，这表明闭包可以支持多个自由变量的形式。

    03. lambda表达式-匿名函数

    3.1 定义和语法

    • lambda 表达式是定义一个简单的函数。
    • 用来表示函数内部只有一个表达式。
    • lambda函数拥有自己的命名空间，且不能访问自有参数列表之外或全局命名空间里的参数。
    变量名 = lambda 参数1...参数n:表达式

    3.2 lambda表达式和普通函数

    • 如果要计算两个数相乘，普通函数的写法是
    def func(a, b):
    return a * b

print(func(2, 3))
>>>
6
    • 使用lambda表达式，试函数更简单
    num1 = (lambda a, b: a * b)
print(num1(2, 3))
>>>
6

    3.3 匿名函数下的if表达式

    isadulthood = lambda age:"已成年" if (age>18) else "未成年"

print("小明的年龄是19岁，小明:"+isadulthood(19))
>>>
小明的年龄是19岁，小明:已成年

    3.4 无参数的匿名函数

    a=lambda :"hello"
print(a())
>>>
hello

    等价函数是：

    def a():
    return "hello"

print(a())
>>>
hello

    3.5 匿名函数的优点

    • 对于单行函数，使用 lambda 表达式可以省去定义函数的过程，让代码更加简洁；
    • 对于不需要多次复用的函数，使用 lambda 表达式可以在用完之后立即释放，提高程序执行的性能。
