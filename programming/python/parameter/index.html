    函数传参

    01. 形参和实参

    1.1 基本定义

    • 形参：在定义函数时，函数名后面括号中的参数就是形式参数，例如:
    # 在定义函数时，这里的object就是形参
def demo(object):
    print(object)
    • 实参：在调用函数时，函数名后面括号中的参数为实参，例如：
    # 此时，a表示实参
a = "hello python"
demo(a)

    实参和形参的区别，就如同剧本选主角，剧本中的角色相当于形参，而演角色的演员就相当于实参。

    1.2 值传递和引用传递

    • 值传递： 适用于实参类型为不可变类型（字符串、数字、元组）；
    • 引用（地址）传递：适用于实参类型为可变类型（列表，字典）；

    值传递和引用传递的区别是，函数参数进行值传递后，若形参的值发生改变，不会影响实参的值；而函数参数继续引用传递后，改变形参的值，实参的值也会一同改变。

    def demo(obj) :
    obj += obj
    print("形参值为：",obj)
print("-------值传递-----")
a = "C语言中文网"
print("a的值为：",a)
demo(a)
print("实参值为：",a)
>>>
-------值传递-----
a的值为： C语言中文网
形参值为： C语言中文网C语言中文网
实参值为： C语言中文网

print("-----引用传递-----")
a = [1,2,3]
print("a的值为：",a)
demo(a)
print("实参值为：",a)
>>>
-----引用传递-----
a的值为： [1, 2, 3]
形参值为： [1, 2, 3, 1, 2, 3]
实参值为： [1, 2, 3, 1, 2, 3]

    值传递实质上 是将实参的副本传递给参数，在内存中，新建了一个地址，因此不会改变原有对象的值； 引用传递本质上也是通过值传递的方式，只是在函数内部修改了实参的变量名的地址引用。

    02. 传递参数

    2.1 位置传参

    位置传参，即按照定义函数时设置的形参，按顺序和数量一一对应。

    • 位置传参要按照函数形参顺序传递，否则程序运行结果会出乎意料，但不会报错；
    • 同时需要传递的实参和形参数量需要保持一致，否则解释器会报错。
    def people(name, age, weight):
    return "姓名：{}\n年龄：{}\n体重：{}".format(name,age, weight)

xiaoming = people("xiaoming", 19, 120)
print(xiaoming)
>>>
姓名：xiaoming
年龄：19
体重：120
    • 参数顺序不同导致的错误：
    def people(name, age, weight):
    return "姓名：{}\n年龄：{}\n体重：{}".format(name, age, weight)

xiaoming = people(19, "小明", 120)
print(xiaoming)
>>>
姓名：19
年龄：小明
体重：120
    • 数量不同导致的错误：
    def people(name, age, weight):
    return "姓名：{}\n年龄：{}\n体重：{}".format(name, age, weight)

xiaoming = people("小明", 120)
print(xiaoming)
>>>
TypeError: people() missing 1 required positional argument: 'weight'

    2.2 关键字传参

    关键字传参是指在调用函数式，按照函数内的形参名来确定输入实参值。

    • 关键字传参可以不考虑位置的顺序，只要指定形参-实参对 对应就可以
    • 关键字传参和位置传参可以混用，但需要注意，混用的情况，关键字传参必须在所有位置传参结束后
    def people(name, age, weight):
    return "姓名：{}\n年龄：{}\n体重：{}".format(name, age, weight)

xiaoming = people("小明",age=18,weight=120)
print(xiaoming)
print("----分割线----")
xiaomei = people(name="小美", age=20,weight=90)
print(xiaomei)

    上述实例的运行结果是：

    姓名：小明
年龄：18
体重：120
----分割线----
姓名：小美
年龄：20
体重：90

    以上实例中，小明的实参传递混用了位置传参和关键字传参。格式需要保持关键字传参需要在位置传参之后

    2.3 默认值

    在定义函数时，如果某个形参为常用值，不需要每次调用函数都传递，则可以定义默认值。语法如下：

    def 函数名(...，形参名，形参名=默认值)：
    代码块

    在定义默认值时需注意，将默认值放在最后。

    • 默认值的定义和调用演示：
    def people(name, age, weight=100):
    return "姓名：{}\n年龄：{}\n体重：{}".format(name, age, weight)

xiaoming = people("小明",age=18)
print(xiaoming)
>>>
姓名：小明
年龄：18
体重：100
    • 默认值的修改：如果函数内的默认值无法满足调用需求，可以以关键字传参的方式更改默认值：
    xiaomei = people("小美",18,weight=120)
print(xiaomei)
>>>
姓名：小美
年龄：18
体重：120
    • 查看函数默认值：使用__defaults__ 可以查看函数内的默认值，返回是一个元组：
    print(people.__defaults__)
>>>
(100,)

    2.4 传递任意数量的实参

    有时可能需要一个函数，能够处理的参数的个数是不确定的，这个时候，可以使用多值参数。多值参数有两种：

    • *args ：定义参数时，变量名前加一个* 或直接以*args 命名，此参数用来接收元组；
    • **kwargs : 定义参数时，变量名前加两个* 或直接以*kwargs 命名，此参数用来接收字典；
    def demo(num, *args, **numbers):
    print(num)
    print(args)
    print(numbers)

demo(1, 2, 3, 4, 5, name="xiaoming", age=18)
>>>
1
(2, 3, 4, 5)
{'name': 'xiaoming', 'age': 18}

    在调用带有多值函数时，如果需要将元组或字典直接传递给函数，可以使用多值参数的拆包：

    • 在元组变量前加一个*
    • 在字典变量前加两个*
    def demo(num, *args, **numbers):
    print(num)
    print(args)
    print(numbers)

num1 = 10
tup1 = ("宝马", "奔驰", "奥迪")
dict1 = {"name": "小明", "age": 18}
demo(num1,*tup1,**dict1)
>>>
10
('宝马', '奔驰', '奥迪')
{'name': '小明', 'age': 18}
