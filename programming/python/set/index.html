    集合

    01.集合的定义

    • 集合(set)是一个无序的不重复序列
    • 集合只能存储不可变数据类型，包括整形、浮点型、字符串、元组。无法存储字典和列表。
    • 使用{} 包括，相邻元素以, 分割。
    • 集合中的元素是唯一的，如果存在重复元素，只会保留一份。
    • 由于集合是无序的，每次输出排序顺序可能都不相同。

    1.1 {} 方法创建集合

    a = {1, 2, 3}
>>>
{1, 2, 3}
b = {1, 2, 1, (1, 2)}
>>>
{1, 2, (1, 2)}

    1.2 set() 方法创建集合

    a = set(("hello", 1 ,2,(1,2)))
>>>
{(1, 2), 1, 2, 'hello'}

b = set([1, 2, 3, ("a", "b")])
>>>
{1, 2, 3, ('a', 'b')}

c = set("hello")
>>>
{'e', 'l', 'h', 'o'}

    set()内传递一个参数，如果是多个元素，可以使用元组或者列表进行传参。

    02. 集合的基本操作

    2.1 访问元素

    • 由于集合中的元素是无序的，所以不能像列表一样使用下标的方式访问元素。因此，访问集合中的数据最常用的方式是使用循环遍历
    a = set("hello")
for i in a:
    print(i, end="")
>>>
ehol
    • 另一种方式是使用in 判断元素是否在集合内
    a = set("hello")
print("o" in a)
>>>
True

    2.2 删除元素

    删除集合元素有几种常用的方法。remove、discard 、pop 、clear

    • remove()方法用户删除集合中指定元素。但如果指定元素不存在，解释器会报错
    a = set("hello")
a.remove("l")
print(a)
a.remove("l")
print(a)

>>>
{'e', 'h', 'o'}
KeyError: 'l'
    • discard() 方法和remove方法使用完全一样，区别是，如果元素不存在，不会报错。
    a = set("hello")
a.remove("l")
print(a)
a.discard("l")
print(a)
>>>
{'h', 'e', 'o'}
{'h', 'e', 'o'}
    • pop() 方法可以随机删除集合中的一个元素。
    a = {1, 2, 3}
a.pop()
print(a)
>>>
{2, 3}
    • clear() 方法用于清空集合。
    a = {1, 2, 3}
a.clear()
print(a)
>>>
set()

    2.3 添加元素

    • add() 方法向集合内添加元素，如果元素已存在，则不进行任何操作。只能添加不可变数据类型。
    a = {1,2,3}
a.add((3,4))
print(a)
>>>
{1, 2, 3, (3, 4)}
    • update() 方法向集合添加元素，可以添加列表和字典。
    a = {1,2,3}
a.update([3,4])
print(a)
>>>
{1, 2, 3, 4}

    2.4 交集、并集、差集和对称差集运算

    • 集合最常做的操作就是进行交集、并集、差集以及对称差集运算。图中有两个集合，set1={1,2,3}和set2={3,4,5}。

    运算操作 Python运算符 含义 例子
    交集 & 取两集合公共的元素 >>> set1 & set2
    {3}
    并集 取两集合全部的元素
    差集 - 取一个集合中另一集合没有的元素 >>> set1 - set2
    {1,2}
    >>> set2 - set1
    {4,5}
    对称差集 ^ 取集合 A 和 B 中不属于 A&B 的元素 >>> set1 ^ set2
    {1,2,4,5}

    03. 集合的方法

    ipython3中输入dir(set)。提示集合有以下方法可以使用：

    >>> dir(set)
['add', 'clear', 'copy', 'difference', 'difference_update', 'discard', 'intersection', 'intersection_update', 'isdisjoint', 'issubset', 'issuperset', 'pop', 'remove', 'symmetric_difference', 'symmetric_difference_update', 'union', 'update']

    各个方法的具体语法和结构及功能如下：

    方法名 语法格式 功能 实例
    add() set1.add() 向 set1 集合中添加数字、字符串、元组或者布尔类型 >>> set1 = {1,2,3}
    >>> set1.add((1,2))
    >>> set1
    {(1, 2), 1, 2, 3}
    clear() set1.clear() 清空 set1 集合中所有元素 >>> set1 = {1,2,3}
    >>> set1.clear()
    >>> set1
    set()

    set()表示空集合，{}表示的是空字典
    copy() set2 = set1.copy() 拷贝 set1 集合给 set2 >>> set1 = {1,2,3}
    >>> set2 = set1.copy()
    >>> set1.add(4)
    >>> set1
    {1, 2, 3, 4}
    >>> set2
    {1, 2, 3}
    difference() set3 = set1.difference(set2) 将 set1 中有而 set2 没有的元素给 set3 >>> set1 = {1,2,3}
    >>> set2 = {3,4}
    >>> set3 = set1.difference(set2)
    >>> set3
    {1, 2}
    difference_update() set1.difference_update(set2) 从 set1 中删除与 set2 相同的元素 >>> set1 = {1,2,3}
    >>> set2 = {3,4}
    >>> set1.difference_update(set2)
    >>> set1
    {1, 2}
    discard() set1.discard(elem) 删除 set1 中的 elem 元素 >>> set1 = {1,2,3}
    >>> set1.discard(2)
    >>> set1
    {1, 3}
    >>> set1.discard(4)
    {1, 3}
    intersection() set3 = set1.intersection(set2) 取 set1 和 set2 的交集给 set3 >>> set1 = {1,2,3}
    >>> set2 = {3,4}
    >>> set3 = set1.intersection(set2)
    >>> set3
    {3}
    intersection_update() set1.intersection_update(set2) 取 set1和 set2 的交集，并更新给 set1 >>> set1 = {1,2,3}
    >>> set2 = {3,4}
    >>> set1.intersection_update(set2)
    >>> set1
    {3}
    isdisjoint() set1.isdisjoint(set2) 判断 set1 和 set2 是否没有交集，有交集返回 False；没有交集返回 True >>> set1 = {1,2,3}
    >>> set2 = {3,4}
    >>> set1.isdisjoint(set2)
    False
    issubset() set1.issubset(set2) 判断 set1 是否是 set2 的子集 >>> set1 = {1,2,3}
    >>> set2 = {1,2}
    >>> set1.issubset(set2)
    False
    issuperset() set1.issuperset(set2) 判断 set2 是否是 set1 的子集 >>> set1 = {1,2,3}
    >>> set2 = {1,2}
    >>> set1.issuperset(set2)
    True
    pop() a = set1.pop() 取 set1 中一个元素，并赋值给 a >>> set1 = {1,2,3}
    >>> a = set1.pop()
    >>> set1{2,3}
    >>> a1
    remove() set1.remove(elem) 移除 set1 中的 elem 元素 >>> set1 = {1,2,3}
    >>> set1.remove(2)
    >>> set1
    {1, 3}
    >>> set1.remove(4)
    Traceback (most recent call last):  File "", line 1, in     set1.remove(4)KeyError: 4
    symmetric_difference() set3 = set1.symmetric_difference(set2) 取 set1 和 set2 中互不相同的元素，给 set3 >>> set1 = {1,2,3}
    >>> set2 = {3,4}
    >>> set3 = set1.symmetric_difference(set2)
    >>> set3
    {1, 2, 4}
    symmetric_difference_update() set1.symmetric_difference_update(set2) 取 set1 和 set2 中互不相同的元素，并更新给 set1 >>> set1 = {1,2,3}
    >>> set2 = {3,4}
    >>> set1.symmetric_difference_update(set2)
    >>> set1
    {1, 2, 4}
    union() set3 = set1.union(set2) 取 set1 和 set2 的并集，赋给 set3 >>> set1 = {1,2,3}
    >>> set2 = {3,4}
    >>> set3=set1.union(set2)
    >>> set3
    {1, 2, 3, 4}
    update() set1.update(elem) 添加列表或集合中的元素到 set1 >>> set1 = {1,2,3}
    >>> set1.update([3,4])
    >>> set1
    {1,2,3,4}
