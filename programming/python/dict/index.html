    字典

    01. 字典的定义

    • dictionary（字典） 是 除列表以外 Python 之中 最灵活 的数据类型
    • 字典同样可以用来 存储多个数据
      • 通常用于存储 描述一个 物体 的相关信息
    • 和列表的区别
      • 列表 是 有序 的对象集合
      • 字典 是 无序 的对象集合
    • 字典用 {} 定义
    • 字典使用 键值对 存储数据，键值对之间使用 , 分隔
      • 键 key 是索引
      • 值 value 是数据
      • 键 和 值 之间使用 : 分隔
      • 键必须是唯一的
      • 值 可以取任何数据类型，但 键 只能使用 字符串、数字或 元组

    1.1 创建字典

    • 使用{}创建简单的字典：
    scores = {"语文": 100, "数学":98, "英语": 30}
    • 使用字符串、数字、元组作为字典的key：
    # 字符串
str_dict = {"name": "张三", "age": 18}
# 数字作为key，列表作为值
number_dict = {18:"少年", 19:20, 20:[1, 2, 3]}
# 元组
tup_dict = {(1, 2):"小明", ("西瓜", "黄瓜"):"蔬果"}

    • 使用dict() 创建字典

      • 可以以多个元组或者列表作为参数向dict() 内传递
      num_List = [("小明", "18"), ("小李", 19)]
dict1 = dict(num_List)
print(dict1)
>>>
{'小明': '18', '小李': 19}
      • 如果不向dict() 内传参，则创建了空字典，然后使用dict() 指定关键字创建字典：
      • 通过dict通过关键字创建字典时，字符串形式的key不需要以“” 包裹
      name_dict = {}
print(name_dict)
>>>
{}

name_dict = dict(姓名 = "小明", 年龄 = 19)
print(name_dict)
>>>
{'name': '小明', '年龄': 19}

    1.2 hash

    • Python 中内置有一个名字叫做 hash(o) 的函数
      • 接收一个 不可变类型 的数据作为 参数
      • 返回 结果是一个 整数
    • 哈希 是一种 算法，其作用就是提取数据的 特征码（指纹）
      • 相同的内容 得到 相同的结果
      • 不同的内容 得到 不同的结果
    • 在 Python 中，设置字典的 键值对 时，会首先对 key 进行 hash 已决定如何在内存中保存字典的数据，以方便 后续 对字典的操作：增、删、改、查
      • 键值对的 key 必须是不可变类型数据
        • 字符串
        • 数字
        • 元组
      • 键值对的 value 可以是任意类型的数据

    02. 字典增改查

    2.1 查看字典的值

    • 和列表不同，字典是以key作为索引
    • 字典[key] key使用中括号包裹
    name_dict = {'name': '小明', '年龄': 19, '身高': 1.75}
print(name_dict["年龄"])
>>>
19

    2.2 修改字典的值

    • 字典中不允许key相同，如果存在相同的key，只有后一个值会被记住
    name_dict = {'name': '小明', '年龄': 19, 'name': '小李'}
print(name_dict["name"])
>>>
小李
    • 指定字典的key即可修改对应的值
    name_dict = {'name': '小明', '年龄': 19, '身高': 1.75}
name_dict["身高"] = 1.85
print(name_dict)
>>>
{'name': '小明', '年龄': 19, '身高': 1.85}

    2.3 新增字典的值

    • 指定字典的key，如果key不存在，即新增
    • 为指定的key赋值
    name_dict = {'name': '小明', '年龄': 19, '身高': 1.75}
name_dict["体重"] = 120
print(name_dict)
>>>
{'name': '小明', '年龄': 19, '身高': 1.75, '体重': 120}

    03. 复制和嵌套

    3.1 字典复制

    • 如果想得到字典的副本，不能使用dict2= dict1 的方法复制，因为dict2 只是对dict1 的引用，dict1 更改也将在dict2 更改。
    • 复制字典副本有两种方法，一种是copy 的方法：
      • 语法是新字典名 = 被复制字典名.copy()
    thisdict =  {
  "brand": "Porsche",
  "model": "911",
  "year": 1963
}
mydict = thisdict.copy()
print(mydict)
    • 另一种是使用内建方法dict()
    • 语法：新字典名 = dict(原字典）
    thisdict =  {
  "brand": "Porsche",
  "model": "911",
  "year": 1963
}
mydict = dict(thisdict)
print(mydict)

    3.2 字典嵌套

    • 字典中包含多个字典，称为嵌套字典：
    myfamily = {
  "child1" : {
    "name" : "Phoebe Adele",
    "year" : 2002
  },
  "child2" : {
    "name" : "Jennifer Katharine",
    "year" : 1996
  },
  "child3" : {
    "name" : "Rory John",
    "year" : 1999
  }
}
    • 嵌套已经存在的字典：
    child1 = {
  "name" : "Phoebe Adele",
  "year" : 2002
}
child2 = {
  "name" : "Jennifer Katharine",
  "year" : 1996
}
child3 = {
  "name" : "Rory John",
  "year" : 1999
}

myfamily = {
  "child1" : child1,
  "child2" : child2,
  "child3" : child3
}
    • 嵌套字典的取值：
    dict1[外层key][内层key]

    05. 内置方法

    ipython3中键入dir(dict),可以看到字典有以下方法：

    >>> dir(dict)
['clear', 'copy', 'fromkeys', 'get', 'items', 'keys', 'pop', 'popitem', 'setdefault', 'update', 'values']
>>>
    分类 方法 说明
    删除 clear() 清空字典中的所有元素
    pop() 删除拥有指定键的元素
    popitem() 删除最后插入的键值对
    查找 get() 返回指定键的值
    items() 返回包含每个键值对的元组的列表
    keys() 返回包含字典键的列表
    values() 返回字典中所有值的列表
    复制 copy() 复制字典副本
    fromkeys() 返回拥有指定键和值的字典
    更新 update() 把字典dict2的键/值对更新到dict里
    setdefault() 返回指定键的值。如果该键不存在，则插入具有指定值的键。

    5.1 clear()

    删除字典所有元素

    • 无返回值和参数
    tinydict = {'Name': 'Zara', 'Age': 7}

print ("字典长度 : %d" %  len(tinydict))
>>>
2

tinydict.clear()
print ("字典删除后长度 : %d" %  len(tinydict))
>>>
0

    5.2 pop()

    弹出字典指定键的值

    • 返回被弹出键对应的值
    dict1 = {"name": "xiaoming", "age": 18}
x = dict1.pop("name")
print(x)
>>>
xiaoming
print(dict1)
>>>
{'age': 18}

    5.3 popitem()

    从字典中删除最后一个项目：

    • 返回一个被删除的元组
    dict1 = {"name": "xiaoming", "age": 18}
x = dict1.popitem()
print(x)
print(dict1)
>>>
('age', 18)
{'name': 'xiaoming'}
    • 由于删除的是一个元组，所以可以通过序列解包的方式分别接受被弹出的key 和value
    # 将弹出的元组赋值给a和b
a, b = dict1.popitem()
print(a,b)
>>>
name xiaoming

    5.4 get()

    get() 方法其实就是根据 key 来获取 value，它相当于方括号语法的增强版，当使用方括号语法访问并不存在的 key 时，字典会引发 KeyError 错误；但如果使用 get() 方法访问不存在的 key，该方法会简单地返回 None，不会导致错误。例如如下代码：

    cars = {'BMW': 8.5, 'BENS': 8.3, 'AUDI': 7.9}
# 获取'BMW'对应的value
print(cars.get('BMW')) # 8.5
print(cars.get('PORSCHE')) # None
print(cars['PORSCHE']) # KeyError

    5.5 items()、keys()、values()

    • items() 返回字典所有键值对
    • keys() 返回字典所有的key
    • values() 返回字典所有的值

    使用这三个方法，需要使用list（） 方法将返回值转换为列表再输出 

    price = {"苹果":2, "梨": 2.5, "桔子": 1.9}
# items
items = price.items()
print(list(items))
>>>
[('苹果', 2), ('梨', 2.5), ('桔子', 1.9)]

# keys
keys = price.keys()
print(list(keys))
>>>
['苹果', '梨', '桔子']

# values
values = price.values()
print(list(values))
>>>
[2, 2.5, 1.9]

    5.6 fromkeys()

    fromkeys() 方法使用给定的多个 key 创建字典，这些 key 对应的 value 默认都是 None；也可以额外传入一个参数作为默认的 value。该方法一般不会使用字典对象调用（没什么意义），通常会使用 dict 类直接调用。例如如下代码：

    # 使用列表创建包含2个key的字典
a_dict = dict.fromkeys(['a', 'b'])
print(a_dict) # {'a': None, 'b': None}
# 使用元组创建包含2个key的字典
b_dict = dict.fromkeys((13, 17))
print(b_dict) # {13: None, 17: None}
# 使用元组创建包含2个key的字典，指定默认的value
c_dict = dict.fromkeys((13, 17), 'good')
print(c_dict) # {13: 'good', 17: 'good'}

    5.7 update()

    update方法以一个新字典更新原字典

    • 如果原字典中的键值对已包含需要覆盖的键值对，则会覆盖
    • 如果原字典的键值对不存在，则会新增
    price = {"苹果":2, "梨": 2.5, "桔子": 1.9}
price.update({"苹果": 3, "西瓜": 5.5})
print(price)
>>>
{'苹果': 3, '梨': 2.5, '桔子': 1.9, '西瓜': 5.5}

    5.8 setdefault()

    setdefault 方法根据指定key，查找对应的值。

    • 如果查找的key不存在，会新建键值对，默认值为none。可以指定值
    • 查找的key已存在，即使指定值也不会覆盖
    price = {"苹果":2, "梨": 2.5, "桔子": 1.9}
# key不存在时，添加键值对，默认为none，如果指定值，则添加对应键值对
print(price.setdefault("南瓜", 3.8))
print(price)
>>>
3.8
{'苹果': 2, '梨': 2.5, '桔子': 1.9, '南瓜': 3.8}

# key存在，不会覆盖
print(price.setdefault("苹果"))
print(price.setdefault("苹果",10))
print(price)
>>>
2
2
{'苹果': 2, '梨': 2.5, '桔子': 1.9, '南瓜': 3.8}
