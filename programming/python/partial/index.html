    偏函数

    💡 偏函数是对原函数的二次封装，将现有函数的部分参数预先设定为指定值，从而得到新的函数。相比原函数，偏函数具有较少的参数，更方便调用

    01. 语法介绍

    定义偏函数，需使用 partial 关键字（位于 functools 模块中）

    from functools import partial
偏函数名 = partial(func, *args, **kwargs)

    02. 偏函数的用法1-关键字方式设置偏函数

    from functools import partial

def add(a,b):
    c = a +b
    d = a *b
    return c, d

demo = partial(add,a=1)

print(demo(b = 3))
>>>
(4, 3)

    使用偏函数定义了新的函数后，调用偏函数时，对于剩余的参数，必须使用关键字传参，否则解释器会错误理解传参，造成异常

    03. 偏函数的用法2-位置传参方式设置偏函数

    from functools import partial

def add(a,b):
    c = a +b
    d = a *b
    return c,d

demo = partial(add,2)

print(demo(b = 3))
>>>
(5,6)

    在实际应用中，可以使用一个原函数，然后将其封装多个偏函数，在调用函数时全部调用偏函数，一定程序上可以提高程序的可读性。
