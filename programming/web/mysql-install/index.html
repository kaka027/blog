    mysql安装

    1. Mac安装

    下载地址：MySQL :: Download MySQL Community Server (Archived Versions)

    1.1 系统变量配置

    VIM ~/.bash_profile

    export PATH=${PATH}:/usr/local/mysql/bin

    :wq  -->表示退出编辑模式并保存

    source ~/.bash_profile  -->将mysql添加到系统变量
    • 以上方法每次打开终端都需要输入source ~/.bash_profile ；一劳永逸的方法是:
    vi ~/.zshrc

export PATH=${PATH}:/usr/local/mysql/bin

:wq

    source ~/.zshrc

    1.2 登录模式修改密码

    set password = password("newpassword")

    2. Win安装

    2.1 安装补丁

    • vcredist_64.exe
    • dxwebsetup.exe

    2.2 安装

    • 将下载后的mysql解压至安装目录

    2.3 创建配置文件

    • 在安装目录下新建my.ini文件

    2.4 初始化

    • 打开终端 & 以管理员的权限去运行

    • 输入初始化命令
    "D:\Program Files (x86)\mysql\bin\mysqld.exe"  --initialize-insecure

    2.5 启动mysql

    • 制作服务
    "D:\Program Files (x86)\mysql\bin\mysqld.exe"  --install mysql57
    • 启动和停止
    >>> net start mysql57
>>> net stop mysql57
    • windows桌面窗口启动
    服务-mysql57-启动/停止

    2.6 添加系统变量

    高级系统设置-->系统变量-->path-->添加D:\Program Files (x86)\mysql\bin

    2.7 在cmd启动

    >>> mysql -u root -p

    2.8 设置密码

    • 第一次登陆执行完启动命令后不输入密码直接回车，然后设置密码
    set password = password('newpassword');
