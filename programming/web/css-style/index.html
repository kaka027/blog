    CSS样式

    1. 高度和宽度

    .c1{
    height: 300px;
    width: 50%;
}
    • 宽度设置支持像素和百分比；
    • 宽高设置对行内标签默认无效；
    • 对块级标签默认有效（但块级标签非常霸道，即时右边空出来也不可以使用）

    2. 行内&块级特性

    • 解决块级标签独占一行，行内标签无法改变宽高的问题
    • display: inline-block 改变标签特性，使其既具备行内标签的特性，又具备块级标签的特性
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .c1{
            display: inline-block;
            height: 300px;
            width: 300px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
   <div class="c1">中国联通div</div>
   <span class="c1">中国联通span</span>
   <div class="c1">中国联通div</div>
   <span class="c1">中国移动span</span>
</body>
</html>

    2.1 行内和块级转换

    # 块级标签转为行内
<div style="display: inline">中国联通div</div>

# 行内转块级
<span style="display: block">中国联通span</span>

    3.字体和颜色

    .c1{
      color: red;  <! 字体颜色
      font-size: 34px; <! 字体大小
      font-weight: 500; <!字体粗度
      font-family: Lantinghei SC; <!字体
  }

    4.文字对齐

    • text-align:center 水平方向居中；
    • line-height 垂直方向对齐，数值和块高度一致，即垂直居中
      • 当块内有多行文本时，不能这样使用
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .c1{
            width: 50%;
            height: 100px;
            border: 1px solid red;

            text-align:center; /* 水平方向居中*/
            line-height: 100px; /* 垂直方向居中*/
        }
    </style>
</head>
<body>
   <div class="c1">中国联通div</div>
</body>
</html>

    5. 浮动

    • float:right 标签向右边浮动
    <div>
    <span>左边</span>
    <span style="float: right">右边</span>
</div>

    5.1 块级标签浮动

    • 块级标签入div设置float 参数后，将不再是快级标签，会根据自身设置的宽高占位
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .item{
            float:left;
            width: 300px;
            height: 100px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
    <div>
        <div class="item"></div>
        <div class="item"></div>
        <div class="item"></div>
        <div class="item"></div>
    </div>
</body>
</html>

    5.2 脱离文档流

    • 如果标签设置float 样式，需要在该标签的末尾添加clear:both 属性，否则改标签会脱离文档流流飞起来，无法撑起父标签
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .item{
            float:left;
            width: 300px;
            height: 100px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
    <div style="background-color: blue">
        <div class="item"></div>
        <div class="item"></div>
        <div class="item"></div>
        <div class="item"></div>
        <div style="clear: both"></div>
    </div>
</body>
</html>

    6. 内边距

    • 块内元素与块边缘的距离，如果设置了内边距，块大小会被撑开
    • padding-top: 顶部
    • padding-left: 左边
    • padding-right: 右边
    • padding-bottom: 底部
    • padding:20px上下左右全是20px的内边距
    • padding:20px 10px 5px 20px 上、右、下、左内边距
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
       .c1{
           width: 200px;
           height: 200px;
           border: 1px solid red;

           padding-top: 20px;      顶部
           padding-left: 20px;     左边
           padding-right: 20px;    右边
           padding-bottom: 20px;   底部
       }
       .c2{
           background-color: gold;
       }
    </style>
</head>
<body>
    <div class="c1">
        <div class="c2">恭喜发财</div>
        <div>万事如意</div>
    </div>
</body>
</html>

    7. 外边距

    body默认有外边距

    • 自身于相邻块的边距
    • margin-top margin-left margin-right margin-bottom 上 左 右 下
    • margin-top 四边
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
       .c1{
           height: 200px;
           background-color: gold;
       }
       .c2{
           height: 100px;
           background-color: red;
           margin-top: 20px;
       }
    </style>
</head>
<body>
    <div class="c1"></div>
    <div class="c2"></div>
</body>
</html>

    8.居中

    • 文本居中
    <div style="text-align: center">你好</div>
    • 区域居中（自己要有宽度）
      • margin:0 auto 表示上下0 左右居中
    <div style="width: 300px;margin: auto " >你好</div>

    12. hover（伪类）

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .c1 {
            color: red;
            font-size: 18px;
        }

        .c1:hover {
            color: green;
            font-size: 50px;
        }

        .c2 {
            height: 300px;
            width: 500px;
            border: 3px solid red;
        }

        .c2:hover {
            border: 3px solid green;
        }

        .download {
            display: none;
        }

        .app:hover .download {
            display: block;
        }
        .app:hover .title{
            color: red;
        }
    </style>
</head>
<body>
<div class="c1">联通</div>
<div class="c2">广西</div>

<div class="app">
    <div class="title">下载APP</div>
    <div class="download">
        <img src="images/qcode.png" alt="">
    </div>
</div>

</body>
</html>

    13. after

    13.1 after拼接

            <style>
        .c1:after{
            content: "你好";
        }
    </style>
</head>
<body>
<div class="c1">联通</div>
<div class="c2">广西</div>

    13.2 after脱离文档流

    如果标签使用float 样式，会使标签脱离文档流，无法撑开父元素，有两种解决办法

    • 新建标签中添加clear:both
    • 使用after 的clearfix
    .clearfix:after{
        content:"";
        display:block;
        clear:both;
}

    14. position

    14.1 fixed-固定在窗口某个位置

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>

        .back{
            position: fixed;
            width: 60px;
            height: 60px;
            border: 1px solid red;

            right: 10px;
            bottom: 50px;
        }
    </style>
</head>
<body>

<div style="height: 1000px;background-color: #5f5750"></div>

<div class="back"></div>

</body>
</html>

    14.2 relative和absolute

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>

        .c1{
            height: 300px;
            width: 500px;
            border: 1px solid red;
            margin: 100px;

            position: relative;
        }
        .c1 .c2{
            height: 59px;
            width: 59px;
            background-color: #00FF7F;

            position: absolute;
            right: 20px;
            bottom: 10px;
        }
    </style>
</head>
<body>
    <div class="c1">

        <div class="c2"></div>

    </div>
</body>
</html>
