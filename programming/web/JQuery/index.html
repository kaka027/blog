    • js
    // 实现点击后展开不自动闭合，再次点击后闭合
function clickMe(self){
    var hasHide = $(self).next().hasClass("hide");
    if (hasHide){
        $(self).next().removeClass("hide");
    }else {
        $(self).next().addClass("hide");
    }
}

    4.2 自动闭合

    点击后展开，当点击其他菜单，第一次被展开的自动闭合

    //点击后展开，当点击其他菜单，第一次被展开的自动闭合
function clickMe(self){
    // 让自己下面的功能点击后展示出来
    $(self).next().removeClass("hide");
    // 找自己父亲、父亲的所有兄弟、寻找class=content、添加hide样式
    $(self).parent().siblings().find(".content").addClass("hide");
}

    4.3 案例知识点

    • css内cursor: pointer 表示鼠标放上去变成手行；
    • css内display: none 表示隐藏、不展示
    • onclick=func(this) 点击自己，在js代码中向函数中传递自己$(self)
    • func(self) 函数设置参数，self是参数名，当运行函数时，向self传递的参数是this ，即自己这个标签
    • hasClass 是否有参数内这个样式
    • removeClass 删除参数内样式
    • addClass 增加参数内样式

    • js部分
    function getInfo(){
    //1. 获取用户输入的姓名和邮箱
    var userName= $("#txtUser").val();
    var email = $("#txtEmail").val();

    //2. 字符串拼接
    var newString = userName + "-" + email

    //3.创建li标签,并写入内容
    var newLi = $("<li>").text(newString);

    //4. 把新创建的li标签放在ul里面
    $("#view").append(newLi);
}

    6.3 页面框架加载完就开始执行js

    • 正常的js写法，是需要html标签全部加载完后才会开始执行js代码，如果页面存在大图片等情况，效率会低
    • jQuery的$(function(){} 作用域内，表示当html框架加载完就开始执行，页面已经有了对应的标签，无需等待标签内元素加载完成
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<ul>
    <li>百度</li>
    <li>谷歌</li>
    <li>搜狗</li>
</ul>

<script src="static/jquery-3.6.0.min.js"></script>
<script>
    $(function () {

        // 当页面的框架加载完成之后，自动就执行。
        $("li").click(function () {
            $(this).remove();
        });

    });
</script>
</body>
</html>
