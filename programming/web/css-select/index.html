    CSS选择器

    1. css应用方式

    1.1 在标签上直接使用

    <img src="..." style="height:100px" />

<div style="color:red;">中国联通</div>

    1.2 在head标签中写样式

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .c1{
            color:red;
        }
    </style>
</head>
<body>

<h1 class='c1'>用户登录</h1>
<h1 class='c1'>用户登录</h1>
<h1 class='c1'>用户登录</h1>
<h1 class='c1'>用户登录</h1>

</body>
</html>

    1.3 在新文件中写样式

    • common.css
    .c1{
    height:100px;
}

.c2{
    color:red;
}
    • html文件中引入
      • rel=“stylesheet” 引入外部样式表
      • href=”css文件的路径”
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <link rel="stylesheet" href="common.css" />
</head>
<body>

<h1 class='c1'>用户登录</h1>
<h1 class='c2'>用户登录</h1>
<h1 class='c2'>用户登录</h1>
<h1 class='c1'>用户登录</h1>

</body>
</html>

    2. css选择器

    2.1 ID选择器

    • ID在body内是唯一的
            <style>
        #c1{
            color: red;
        }
    </style>

    <h1 id="c1">中国联通<h1>

    2.2 类选择器

    • class可以复用到多个标签
    .c1{
    color: green;
}

<div class="c1">今天天气真好啊</div>

    2.3 标签选择器

    • 选中页面所有指定的标签
    <head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        li{
            background-color: blue;
        }
    </style>
</head>
<body>
    <ul>
        <li>百度</li>
        <li>腾讯</li>
        <li>谷歌</li>
    </ul>
</body>

    2.4 属性选择器

    • 根据标签指定属性
    <head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        input[type="text"]{
            border: 1px solid red;
        }
    </style>
</head>
<body>
    <input type="text">
    <input type="password">
</body>
    • 根据类指定属性
    <head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .v1[name="123"]{
            color: red;
        }
    </style>
</head>
<body>
   <div class="v1" name="123">中国移动</div>
   <div class="v1" name="456">中国联通</div>
   <div class="v1" name="999">中国电信</div>
</body>

    2.5 后代选择器

    • 所有后代以空格隔开
    • 指定后代以> 分隔
    <head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .yy li{
            color: red;
        }
        .yy>div>a{
            color: blue;
        }
    </style>
</head>
<body>
   <div class="yy">
      <a>百度</a>
      <div>
          <a>谷歌</a>
      </div>
      <ul>
          <li>美国</li>
          <li>日本</li>
          <li>韩国</li>
      </ul>
</body>

    3. 多个样式覆盖问题

    • 当标签使用多个类，该标签同时使用多个css的定义
      • 如果存在属性重复，则style部分，下面代码部分会覆盖上面部分，即重复时使用下面的样式。覆盖逻辑和标签内class指定的顺序无关
      • 如果存在重复属性时，不希望被覆盖，依然使用上面的样式，可以在该属性后面加上!important 表示重要
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .c1{
            color: red !important;
            border: 1px solid red;
        }
        .c2{
            font-size: 28px;
            color: blue;
        }
    </style>
</head>
<body>
   <span class="c1 c2">中国联通</span>
</body>
</html>
