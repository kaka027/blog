    选取数据

    DataFrame数据选取包括三种情况： - 单维选取，仅选取行或列,只能为一个维度设置筛选条件；df[] - 多维选取（区域选取），.loc[] .iloc[] df.ix[] - 单元格选取，准确定位一个单元格：df.at[] df.iat[]

    import pandas as pd
import numpy as np
data = {'name': ['Joe', 'Mike', 'Jack', 'Rose', 'David', 'Marry', 'Wansi', 'Sidy', 'Jason', 'Even'],
        'age': [25, 32, 18, np.nan, 15, 20, 41, np.nan, 37, 32],
        'gender': [1, 0, 1, 1, 0, 1, 0, 0, 1, 0],
        'isMarried': ['yes', 'yes', 'no', 'yes', 'no', 'no', 'no', 'yes', 'no', 'no']}
labels = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j']
df = pd.DataFrame(data, index=labels)
df

    1. 行列选取

    1. 选取行

    • 整数索引切片
    • 标签索引切片
    • 布尔数组
    a) 整数索引切片（左闭右开）
    # 选取前两行
df[:2]
    name age gender isMarried
    a Joe 25.0 1 yes
    b Mike 32.0 0 yes 
    # 选取第二行到第四行
df[2:4]
    name age gender isMarried
    c Jack 18.0 1 no
    d Rose NaN 1 yes 
    # 最后3行
df[-3:]
    name age gender isMarried
    h Sidy NaN 0 yes
    i Jason 37.0 1 no
    j Even 32.0 0 no
    b) 标签索引切片（前闭后闭）
    # 第一行
df[:'a']
    name age gender isMarried
    a Joe 25.0 1 yes 
    # a-d
df['a':'d']
    name age gender isMarried
    a Joe 25.0 1 yes
    b Mike 32.0 0 yes
    c Jack 18.0 1 no
    d Rose NaN 1 yes 
    # g-末尾
df['g':]
    name age gender isMarried
    g Wansi 41.0 0 no
    h Sidy NaN 0 yes
    i Jason 37.0 1 no
    j Even 32.0 0 no
    c）布尔判断
    # 前三行
df[[True, True, True, False, False, False, False, False, False, False]]
    name age gender isMarried
    a Joe 25.0 1 yes
    b Mike 32.0 0 yes
    c Jack 18.0 1 no 
    # age>30的行
# df[[each>30 for each in df['age']]]
df[df['age'] > 30]
    name age gender isMarried
    b Mike 32.0 0 yes
    g Wansi 41.0 0 no
    i Jason 37.0 1 no
    j Even 32.0 0 no 
    # age>30和isMarried=no
# 多条件需要在括号内填条件
df[(df['age'] > 30) & (df['isMarried'] == 'no')]
    name age gender isMarried
    g Wansi 41.0 0 no
    i Jason 37.0 1 no
    j Even 32.0 0 no 
    # 多条件
df[(df['age'] == 20) | (df['age'] == 32) & (df['isMarried'] == 'no')]
    name age gender isMarried

    j Even 32.0 0 no
